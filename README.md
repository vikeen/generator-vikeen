generator-vikeen
================

Web application codebase bootstrap via Yeoman

## Installation
  Install yeoman framework
   - `npm install -g yo`

  Install our generator
   - `npm install -g generator-vikeen`

## Use
  1. Make a new project directory then switch to it
  2. `yo vikeen`
  3. Presto

## Tests

  `npm test`
