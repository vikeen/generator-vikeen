generator-vikeen
================

Web application codebase bootstrap via Yeoman

## Installation
  Install yeoman framework
   - `npm install -g yo`

  Install our generator
   - `npm install -g generator-vikeen`

## Use
  1. `mkdir PROJECT_NAME`
  2. `cd PROJECT_NAME` 
  3. `yo vikeen`
  4. Presto

## Tests

  `npm test`
